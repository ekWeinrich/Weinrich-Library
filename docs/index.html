Einbindung in eine AS-Regel

Damit die Bibliothek lauffähig ist, MÜSSEN folgende ELO-Bibliotheken inkludiert werden:

  • lib_Class
  • ix: IndexServer Functions
  • lib_weinrich.as

Um den vollen Funktionsumfang nutzen zu können, wird empfohlen folgende ELO-Bibliotheken zusätzlich zu inkludieren:

  • fu: File Utils
  • wf: Workflow Utils
  • db: DB Access
  • lib_sol.common.WfUtils
  • lib_sol.common.SordUtils
  • lib_sol.common.RepoUtils
  • lib_sol.common.AclUtils

Logging

Initialisiere die Log-Funktion 1 mal initial am Anfang der Regel: weinrich.as.Utils.initLogging(true, "NameDerAS-Regel"); True/False stehen hier für Debug-Modus. Bei True wird immer ein Logeintrag geschrieben. Bei False nur, falls man später einen Prioritätslog verwendet.

Danach kann folgendermaßen geloggt werden: weinrich.as.Utils.logging(false, "Dies ist ein Dummy-Logeintrag"); True/False stehen hier für Prioritätslog. Bei True wird immer ein Logeintrag geschrieben, auch wenn man nicht im Debug-Modus ist.
